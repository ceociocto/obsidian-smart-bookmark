# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-03-20

### Added
- Initial release
- Support for importing bookmarks from Chrome, Safari, Edge, and Firefox
- Basic content analysis for imported bookmarks
- Automatic generation of structured Obsidian notes
- Multi-language support (English and Chinese)
- Settings page with configuration options
- Search and tag management features
- Reserved API endpoints for future cloud AI services integration
- Plugin icon and UI

### Features
- Parse browser bookmark files from multiple browsers
- Extract website metadata (title, description, keywords)
- Generate well-structured Markdown notes with frontmatter
- Organize bookmarks into folders and collections
- Support for custom note templates
- Progress tracking during import operations
- Error handling and user-friendly messages
